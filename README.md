# Tune-Up-Racing

The source code for my racing game that I have been developing for the past two years or so.

Playable Demo available here:
https://rasmus-02.itch.io/tune-up-racing

# Game Features:

Advanced Vehicle and Engine Customization
- Comprehensive modification system allowing deep mechanical and aesthetic customizations
- Realistic engine swaps with performance impact
- Paint and tuning systems


Vehicle Simulation
- Wear and degradation mechanics
- Arcady car physics to allow for satisfying gameplay
- Simulated customizeable engine, featuring turbos, superchargers, twin charging and much more


Intelligent AI Systems
- Dynamic procedural opponent generation algorithm that generates oponent cars with similar performance to the players car
- AI opponents to race against


In-Game Shops
- Dynamic in-game car marketplace where cars get sold and added constantly (similar to nettiauto)
- Parts Shop where player can buy new parts for their cars
- Inventory manager


Environmental Systems
- day/night cycle with dynamic lighting


Technical Highlights:
- Modular, extensible game architecture
- AI for cars that is able to drive at similar performance level as the player
- Procedural generation algorithms
- Vehicle and Engine simulations with low performance impact
- Dynamic engine sounds
